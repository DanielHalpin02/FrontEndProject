
FitnessTracker is a fitness tracking app built using Ionic and Angular.

It allows users to:
- Log their weight
- View progress on a dashboard
- Browse workouts from an external API
- Search for exercises
  
Features
- Weight logging (add and delete entries)
- Dashboard with latest weight, total entries, and average
- Workout explorer using external API
- Search functionality for workouts
- Workout details page with improved descriptions
- Local data storage using Ionic Storage
  
Technologies I Used
- Ionic Framework
- Angular (Standalone Components)
- TypeScript
- Ionic Storage
- Angular HttpClient
- Capacitor
